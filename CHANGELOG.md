# Changelog

## v0.2.0

### Added
- Added four simplified quick-look presets: Pear, Cozy, Bold, and Gloss.
- Added Hue Shift Speed for animated hue cycling.
- Added shared keyword syncing for Pear material features.
- Added material validation warnings in the custom inspector.
- Added updated demo scene presentation for feature and preset comparisons.

### Changed
- Renamed the original presets from Soft Anime, Cel Shadow, and Glossy Toy to Cozy, Bold, and Gloss.
- Retuned preset values for a more cohesive Pear materials.
- Updated website and README copy for the v0.2 workflow.
- Improved hue shifting so it works more clearly on final material color.

### Known Issues
- Inverted-hull outlines may show artifacts on hard-edged capped meshes such as Unity’s default cylinder.
- URP and Built-In Render Pipeline shaders are separate and may not match pixel-perfectly.
