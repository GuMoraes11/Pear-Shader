Unity URP Tech Art Project

Stylized toon shading with artist-friendly controls.

Pear is a compact Unity URP shader toolkit built for stylized characters, props, and portfolio-ready material demos. It combines hand-written HLSL, a custom Unity material inspector, ramp lighting, rim light, emission, and matcap effects.

Pear is designed like a small production shader tool: readable shader code, clean editor tooling, grouped controls, feature toggles, and a demo scene that shows technical art value instead of only visual output.

Current Features

Built for stylized material workflows.

URP Toon Lighting

Custom ShaderLab/HLSL lighting with shadow color, light threshold, softness, and direct light intensity controls.

Multi-Band Shading

Optional midtone band support for shadow → midtone → lit shading, giving artists more control than a basic two-tone shader.

Ramp Texture Mode

Horizontal ramp textures can define custom lighting bands for more complex stylized looks without adding endless sliders.

Rim Light

Fresnel-style rim lighting adds silhouette highlights for anime, toy-like, or neon material styles.

Emission

Emission color, mask, intensity, and pulse controls support glowing accents and future neon presets.

Matcap Reflection

View-based matcap sampling creates fake glossy, pearlescent, or metallic highlights without full PBR complexity.

Custom Inspector

A shader UI that artists can actually use.

Pear includes a custom Unity ShaderGUI with grouped foldouts, feature toggles, a customizable accent color, and clean controls for each stylized effect.

  • Grouped controls for Base, Toon Lighting, Rim, Emission, Matcap, and Advanced
  • Feature toggles connected to local shader keywords
  • Custom styled inspector header and color accents
  • Hidden controls when optional effects are disabled
